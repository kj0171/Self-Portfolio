1) YATRA | B2B-HOTEL

B2B-Hotel is basically a web portal used by travel agents for hotel bookings. My role in this project was to create rest-microservice's. The complexity in this project was designing a code in such a way to support multiple suppliers , commissioning and marking up the price based on the user (agent) and the supplier. Aerospike was used for uploading special commission against certain hotel's and it was kept dynamic. Java Executor service was used for parallel processing of independent tasks. There were also many features like Hold And Pay , Pan Verification and Mark up along with the normal flow for the hotel booking.

Technology : Java , SpringBoot , Aerospike , Mongo , MySql , Hibernate
